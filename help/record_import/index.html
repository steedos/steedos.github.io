数据导入

如果您的 Excel 表格中已经有初始业务数据，可以快速导入系统中#

  • 在设置-高级设置-数据导入中添加导入任务
  • 设定 Excel 中的列和对象字段的对应关系
  • 上传 Excel 附件，如：联系人.xlsx
  • 点击执行导入
